var-

Variables declared with var are accessible anywhere within the function that they are declared in. They are called Function-scoped.

let-

Variables declared with let are accessible only inside the block where they are declared. This is why They are called Block-scoped.

const-

Variables declared with const are also block-scoped i.e they are accessible only inside the block where they are declared. {Immutable= once assigned the values cannot be reassigned}

null-

It indicates the absence of any object or value in a variable.

undefined-

This indicates to an uninitialized or missing value for a variable. It is considered the default value for the function parameters and anykind of uninitialized variables.

REST API-

A REST API is a way for different devices and servers to communicate with each other over the internet using a set of rules (known as HTTP). It facilitates smooth communication and access to information.
